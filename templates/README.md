# Шаблоны документов

Сюда положите файл шаблона родительского согласия:

- **`parental-consent-template.pdf`** — на него ссылается кнопка «Скачать шаблон»
  в модальном окне согласия (кабинет школьника/лицеиста).

Файл раздаётся статикой Netlify по адресу `/(templates)/parental-consent-template.pdf`.
Пока файла нет — кнопка «Скачать шаблон» вернёт 404. Достаточно положить сюда PDF и запушить.
