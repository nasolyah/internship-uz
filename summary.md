# Стартап: платформа "стартапы ⇄ студенты"

## Идея одной строкой
Платформа, связывающая узбекские стартапы (нужны дешёвые/бесплатные рабочие руки) со студентами и школьниками (нужен реальный опыт + рекомендательное письмо для резюме/поступления).

## Контекст и цель основателя
- Основатель — школьник, планирует поступление в вуз в 2027 году
- Проект нужен не под конкретную приёмную комиссию, а как реальный, доведённый до результата вклад — сильный пункт резюме независимо от того, куда в итоге поступать
- Показывает бизнес-план отцу и будущему партнёру по проекту

## Верификация участников (доверие)

**Студенты** — прогрессивные бейджи: вход через email/Telegram (без барьера) → подтверждён email/Telegram → «студент [вуза]» (университетская почта, опционально) → «подтверждённые навыки» (сертификаты, лучше через API площадок) → «протестирован» (ИИ-тест) → «Verified» (платно, $1–2).

**Стартапы/компании** — риск зеркальный, набор проверок:
- Корпоративная почта (@domain компании) — тот же механизм кода, что и для вузов
- Госреестр юрлиц (my.gov.uz) — сверка названия/ИНН и руководителя
- LinkedIn представителя или активная страница компании в соцсетях
- Ручной созвон с командой платформы — обязателен на старте (первые 10–20 компаний), даёт максимальное доверие

**Несовершеннолетние (10–11 класс, 1–2 курс лицея, ~16–17 лет)** — блокирующая (не опциональная) верификация: возраст/класс указывается при регистрации → если несовершеннолетний — каталог заблокирован до загрузки согласия родителя (готовый шаблон PDF) → ручная проверка документа → доступ открыт. Юридическое основание: по ТК РУз трудовые отношения с лицами до 18 лет требуют письменного согласия родителя; даже при оформлении как «практика» (не трудовой договор) несовершеннолетний имеет ограниченную дееспособность — согласие нужно в любом случае.

## Ключевые решения, принятые в ходе обсуждения

| Вопрос | Решение |
|---|---|
| Верификация студента | Бесплатно: SMS + университетская почта. Платно ($1–2): опциональный upgrade до "verified" статуса |
| Оплата от компаний | На старте — бесплатно (набор базы), позже — premium-подписка $1–2/мес |
| Доп. доход | Реклама на платформе — рассматривается как поздний этап (нужен трафик) |
| Проверка качества | ИИ-тест на навыки + (осторожно) загрузка сертификатов — лучше через API площадок, чем просто сканы |
| Формат MVP | **Не Google Form** — сразу полноценный по виду сайт (лендинг + формы), матчинг вручную на старте, автоматизация позже |
| Юридическая форма отношений | Оформлять как учебную/проектную практику, а не трудоустройство — снимает риски и даёт официальный документ |
| Бренд | Не запускать под чужим именем (обсуждали Impact Admission KZ) — сохранить авторство ради ценности для резюме. Возможное партнёрство по дистрибуции — но без потери бренда |

## Возможности для роста (найдено в ходе исследования)
- **«Вузы — генераторы стартапов»** — с 1 сентября 2026, акселераторы в вузах + IT Park + венчурные фонды
- **Конкурсы «Лучшая стартап-идея» / «Самая инновационная идея»** — гранты от 5 млн до 10 млрд сумов, официальное признание проекта
- **IT Park Uzbekistan — Brand Ambassadors в ВУЗах** — потенциальная точка входа для дистрибуции
- **Impact Admission KZ** и похожие консалтинг-агентства — потенциальные партнёры по дистрибуции (не под их брендом)

## Главные риски
1. Низкое качество исполнителей → подрывает доверие компаний
2. Юридическая неопределённость по неоплачиваемому труду
3. Сложно набрать критическую массу с обеих сторон одновременно
4. Юный возраст основателя может восприниматься как несерьёзность → компенсируется профессиональным оформлением и официальными партнёрствами

## Ближайшие шаги
1. 10–15 валидационных интервью со стартапами и студентами
2. Прототип сайта (лендинг + формы регистрации)
3. Первые 5–10 стартапов и 10–20 студентов для пилота
4. Типовой шаблон соглашения о практике
5. Заявка на ближайший студенческий конкурс стартап-идей
6. Контакт с IT Park Uzbekistan по программе работы со студентами
